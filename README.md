## Development

### Prerequisites

- Node.js 18+ or 20+
- pnpm package manager

### Installation

```bash
pnpm install
```

### Run Development Server

```bash
pnpm dev
```

The site will be available at http://localhost:4321

### Build for Production

```bash
pnpm build
```

The built files will be in the `dist/` directory.
